# Security Policy

## Supported Versions

Es wird die jeweils aktuelle `main`-Version unterstuetzt.

## Reporting a Vulnerability

Bitte melde Sicherheitsprobleme nicht oeffentlich ueber Issues.

Nutze stattdessen die GitHub-Funktion "Report a vulnerability" (Security Advisory) im Repository.

Falls diese Funktion nicht verfuegbar ist, erstelle ein normales Issue ohne technische Details und markiere es klar als sicherheitsrelevant; ein Maintainer wird dann einen vertraulichen Kanal bereitstellen.
